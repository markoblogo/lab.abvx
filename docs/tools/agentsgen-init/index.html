agentsgen init

Bootstrap AGENTS.md + RUNBOOK.md + .agentsgen.json with safe markers (autodetect optional).

CLIPythonBootstrap

What it does

  • Creates the initial AGENTS.md and RUNBOOK.md files for a repo.
  • Can generate baseline config with conservative defaults and autodetection.
  • Starts from marker-owned sections so future updates stay scoped.
