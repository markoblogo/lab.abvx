What it does

  • Lists built-in presets for common stacks (python lib, fastapi, nextjs, monorepo js, cli python, cli node).
  • Bootstraps a repo with a conservative .agentsgen.json without reading docs.
  • Designed to be safe and editable: you can tweak commands after init.
