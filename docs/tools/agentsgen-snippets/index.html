What it does

  • Extracts named snippet blocks from README.md using AGENTSGEN:SNIPPET markers.
  • Generates README_SNIPPETS.generated.md without modifying README itself.
  • Supports --check, --dry-run, --print-diff, and --format json for reviewable workflows.
