Primary role AsciiTheme
Where it fits

Minimal visual utility for readable developer pages and small static sites.

Tags
  • CSS
  • UI Kit
  • Theming

What it does

  • Adds an ASCII visual layer to existing pages with default or ASCII style management.
  • Supports host-theme respect mode or managed light and dark mode with compact injected toggles.
  • Includes reusable base CSS for building small readable sites from scratch in the same visual language.
