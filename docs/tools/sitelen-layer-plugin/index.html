sitelen-layer-plugin

Plugin for sitelen-layer rendering support (toki pona tooling).

PluginToki PonaDev Tool

What it does

  • Adds switchable display layers for existing toki pona text: latin, sitelen pona, and sitelen emoji.
  • Works on static pages and SPA-like pages with mutation observers and route-aware refresh behavior.
  • Supports profile-based activation and mounted toggles for locale-aware integrations.

Quickstart

Basic integration

import { createSitelenLayerPlugin } from 'sitelen-layer-plugin';
import 'sitelen-layer-plugin/styles.css';

createSitelenLayerPlugin({
  container: '#tok-content',
  defaultLayer: 'latin',
  showToggle: true
}).init();
